== Description ==

Transfer Wordpress from one server to another server or Changing The Site URL  is easy now. Configure Wordpress Options Saved in Database. Add a new options or update existing one even you're a non technical person.

Importance

Thousands of wordpress websites transfer from one hosting to another hosting every day. This plugins helps you to configure your website when server changed, website domains changed or remove unwanted options. This plugins helps wordpress developers to take control on wordpress options without phpmyadmin controls.
 

Usage:

It's easy! Once activated, Click on Wp Options links located at left side of the wordpress dashboard.

Notes:

This plugin for WordPress, the publicly available  If you experience any problems with the plugin, feel free to contact me.

This section describes how to install the plugin and get it working.

	2.	Activate the plugin through the 'Plugins' menu in WordPress
